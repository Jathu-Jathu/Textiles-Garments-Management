# 👕 Textiles and Garment Management System

A web-based Textiles and Garment Management System built with PHP, MySQL, HTML, CSS, and JavaScript.

## 🛠️ Tech Stack

- 🌐 HTML
- 🎨 CSS
- ⚡ JavaScript
- 🐘 PHP
- 🗄️ MySQL

##  Features

- 🔐 Secure user authentication
- 👤 Admin dashboard
- 👕 Product management
- 📦 Inventory management
- 🔄 Real-time data updates
- 📱 Responsive user interface
